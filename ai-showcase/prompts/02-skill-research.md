# Prompt: Skill Research & Demand Evidence

**Used for:** Deliverable 2 - Skill Research with demand evidence

---

## Prompt

```
I'm researching skills for ClawFriend Skill Marketplace (Web3 AI Agent platform on BNB Chain). I need 5-10 skill ideas with STRONG demand evidence.

For each skill, provide:

1. **Name**
2. **Target user** - who would pay for this
3. **Problem solved** - the pain point
4. **Demand evidence** - MUST include at least one of:
   - Search volume or keyword data (mention source + date)
   - Existing paid tools in same space (name, pricing)
   - Case studies of successful similar products
   - Academic/industry reports
   - Social proof (follower counts, trending topics)
5. **Current alternatives** - what users use today
6. **How this skill solves it** - specific value proposition
7. **Visibility** - Public / Private / Holder-gated
8. **Pricing** - suggested model
9. **Monetization strategy**

Focus on:
- Web3-native use cases (DeFi, airdrop, security, yield)
- Skills that work with OpenClaw, ElizaOS, Virtuals
- Industrial/utility skills, not pure entertainment

Prioritize skills with verifiable demand — link to real products, news, or data. "AI told me" without source = useless.
```
